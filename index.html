HTML

  • The meta tag contains the infor used by the website
  • The head tag contains meta data
  • The meta tag consists of many functions an important one being the "viewport" function along with the content function which allow the webpage to fit in the screen perfectly
  • Both the meta tag and head tag are functions that do not show on the body of the website
  • The link tag allows us to merge a different type of file within our html code

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside of an element

Git

  • git status: checks the current branch status
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A allows to add all the changes done to the commit
  • git commit -m a commit command with a message of what the commit consists of
  • git pull origin main this command allows us to review our current version of the code to double check if were working on the latest version
  • git push origin branch-name pushes our code to the selected branch
